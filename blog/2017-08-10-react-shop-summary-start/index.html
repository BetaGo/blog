用React构建一个小商城（一）

08月 10日, 2017年

作为一个商城的项目主要的功能有：

  • 登录、登出
  • 查看商品列表，看看单个商品详情。
  • 添加商品到购物车，修改购物车信息。
  • 下订单、结算。
  • 查看订单信息。

根据功能，可能需要这些页面：

  • 登录页
  • 注册页
  • 商品列表页
  • 商品详情页
  • 购物车页
  • 订单信息页
  • 其他页面
